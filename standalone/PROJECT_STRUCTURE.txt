ARCHITECTURE:
The project follows Clean Architecture with dependency inversion:
- Presentation depends on Application
- Application depends on Domain
- Infrastructure depends on Domain (via interfaces)
- Domain has NO dependencies (pure business logic)

This allows:
✓ Easy testing (mock interfaces)
✓ Easy hardware swapping (just implement IUTMDriver)
✓ Easy export format addition (just implement IExportService)
✓ UI framework changes (swap Qt with something else)
