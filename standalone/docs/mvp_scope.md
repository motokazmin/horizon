# MVP Scope

## Included in MVP
- [x] Mock hardware driver (simulated UTM)
- [x] Real-time dashboard with metrics display
- [x] Live stress-strain chart (QtCharts)
- [x] Sample queue management
- [x] SQLite database persistence
- [x] CSV export
- [x] Dark theme UI
- [x] Test start/stop/pause control
- [x] Stress/strain calculations

## Excluded from MVP (Phase 2)
- [ ] Real Tinius Olsen USB driver
- [ ] PDF export
- [ ] Excel export  
- [ ] User authentication/authorization
- [ ] Sensor calibration module
- [ ] Advanced test analysis
- [ ] Multi-language support
- [ ] Network/remote control

## Success Criteria
1. Application launches successfully
2. Can create and run simulated tests
3. Real-time chart updates smoothly
4. Test data saves to database
5. Can export test results to CSV
6. UI matches design mockup
